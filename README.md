# 📝 Python Metin Analiz Aracı

Bu proje, kullanıcıdan alınan bir metni analiz ederek istatistiksel veriler çıkaran basit ve etkili bir Python uygulamasıdır. Python dersi projelerimden biridir.

## 🚀 Özellikler
Uygulama, girilen bir metin üzerinde şu analizleri yapar:
* **Karakter Sayımı:** Metnin toplam uzunluğunu hesaplar.
* **Kelime & Cümle Sayımı:** Metindeki toplam kelime ve cümle sayısını bulur.
* **Kelime Analizi:** En sık kullanılan, en uzun ve en kısa kelimeleri tespit eder.
* **Özel Arama:** Belirli bir kelimenin metin içinde kaç kez geçtiğini raporlar.

## 🛠 Kullanılan Teknolojiler
* **Python 3.x**
* **re (Regex):** Cümle ayıklama işlemleri için.
* **collections.Counter:** Frekans analizi için.
